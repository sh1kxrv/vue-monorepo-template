# Vue Monorepo Template

## Requirements

- Node.js >= 20.10.0
- pnpm >= 9.12.0

## Tools
- Vue 3
- Middleware Router
- Tailwind v4
- Minimalistic Layouts
- Prettier + ESlint (oxlint)

## Features

- Isolated Alias:
You can safely use aliases within pkgs and it will work
